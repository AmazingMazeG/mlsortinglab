Machine Learning Sorting Lab

An interactive game looks at two core Machine Learning ideas:
Classification (supervised learning) and Clustering (unsupervised learning)

Choose a mode

Supervised Learning
This is classification: study a few labeled training examples per class, then predict the class of new, unseen emoji and check your accuracy.
Unsupervised Learning
This is clustering: find the structure yourself, sort the characters into clusters, and explain what feature connects each one.

How many clusters? (k)

You'll get 40 unlabeled Chinese characters as a raw, unlabeled dataset.
Choose k, the number of clusters, drag characters into each one, then explain the rule your clustering follows.
